# Commit Message Rule

|Prefix|Example|
|------|---|
|add: | `add: Makefile and minishell.h file` |
|update: | `update: parse tree generating feat` |
|del: | `del: .swp files` |
|fix: | `fix: execute bug (1)` |
|comment: | `comment: about tokenizer` |
|norm: | `norm: about header files` |
|refact: | `refact: header files` |
